✅ Display name (nickname)

✅ Consent record + timestamp

✅ Device model & manufacturer

✅ Registration IP (AES-256 encrypted, decryptable)

✅ Message count per session
✅ Encrypted messages (decrypted in export)
✅ Flagged/reported message records

✅ All reports filed against user
✅ Reports filed by user
